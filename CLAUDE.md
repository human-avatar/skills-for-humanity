# finnskill

Personal Claude Code skills plugin.

## Skills

Skills live in `skills/`. Each skill has a `SKILL.md` that defines its trigger conditions and instructions.

To add a new skill, create a folder under `skills/` containing a `SKILL.md` file.

## Installation

```bash
claude plugins install /path/to/finnskill
```

## Development

Use the `superpowers:writing-skills` skill when creating or editing skills.
