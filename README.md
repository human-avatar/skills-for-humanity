# skills-for-humanity

**The greatest thinking tools ever developed, available in seconds.**

131 skills · 23 categories · one command to find the right one

---

Not a prompt library. Each skill encodes a complete reasoning methodology — not just what to think, but *how*. A prompt asks Claude to respond. A skill tells Claude how to think.

**Lineage:** de Bono · Meadows · Altshuller · Minto · Goldratt · Tetlock · Klein · Mill · Kant · Aristotle · Noddings · Rawls · Sun Tzu · Clausewitz · Musashi · Nash · Axelrod · Vickrey · Shapley · Spence

---

## Install

```bash
npx @human-avatar/skills-for-humanity
```

Run again to update. Restart Claude Code when done.

---

## Use

```bash
/human   # describe your situation — routed to the right skill automatically
```

Or invoke any category command directly:

```
/logic  /probability  /decision  /constraint  /game-theory
/creativity  /analogy  /play
/communication  /social  /emotional  /ethics  /identity  /narrative  /writing
/systems  /temporal  /historical  /resource  /strategy
/aesthetic  /sensory
```

Every skill pauses before running and asks how deep you want to go.
