# 💡 Creativity

> *The dominant idea is the enemy of the next idea.*

---

## What this category is

The creativity toolkit is the full de Bono suite — twelve structured techniques for generating, exploring, evaluating, and escaping fixed thinking. These aren't brainstorming variations; they're distinct tools with specific jobs:

- Some **generate** options (Alternatives, Concept Fan, Random Entry)
- Some **evaluate** options (PMI, Six Hats, Consider All Factors)
- Some **escape** fixed positions (Lateral Thinking, Assumption Excavator, Provocation)
- Some **explore** territory without judgment (Water Logic, Other Perspectives)
- One **orchestrates** the others for comprehensive sessions (Brainstorm)

The hardest part of creative thinking isn't generating ideas — it's knowing which tool to apply to which situation. The `/s4h-creativity` command handles that.

---

## When to reach for it

- Stuck on a problem and keep returning to the same direction
- About to make a decision and want to make sure you've seen all the options
- Need to evaluate a specific idea fairly, without snap judgment
- Want a comprehensive multi-perspective analysis before committing
- Problem framing itself might be wrong
- Need a jolt from outside the problem space
- Want to think through how others will respond to a decision

---

## When *not* to reach for it

**Creativity tools are for generating and exploring, not for validating.** If you need to know whether something is logically sound, that's [Logic](../logic/). If you need to know whether it's ethical, that's [Ethics](../ethics/).

**Don't reach for creativity when the problem is an information gap.** No amount of creative thinking replaces missing data. Get the information first, then think creatively with it.

The nearest neighbor: [Play](../play/) — which also generates unconventional ideas, but through inversion, reversal, and deliberate absurdity rather than the structured de Bono toolkit.

## Skills in this category

| Skill | What it does | Trigger when... |
|---|---|---|
| **`/s4h-creativity`** | Routes to the right creativity skill for your situation | Not sure which tool fits; describe your situation |
| [`/s4h-creativity-brainstorm`](brainstorm.md) | Orchestrated multi-method session — auto-selects and sequences tools | Big challenge; want the full toolkit applied; "help me think through this" |
| [`/s4h-creativity-assumption-excavator`](assumption-excavator.md) | Surfaces and challenges hidden assumptions in the problem framing | Problem feels intractable; multiple approaches keep failing; framing might be wrong |
| [`/s4h-creativity-lateral-thinking`](lateral-thinking.md) | Escapes dominant patterns to generate genuinely new directions | Stuck on one approach; keep returning to the same idea; "I need to think differently" |
| [`/s4h-creativity-alternatives`](alternatives.md) | Generates all options before evaluating any (APC) | About to decide; want to see the full option set first |
| [`/s4h-creativity-concept-fan`](concept-fan.md) | Expands the solution space at multiple levels of abstraction | Locked into one solution; want to see the full range of approaches |
| [`/s4h-creativity-six-hats`](six-hats.md) | Structured parallel thinking from six distinct angles | Need full multi-perspective analysis; preparing for a meeting; want both optimism and caution |
| [`/s4h-creativity-plus-minus-interesting`](plus-minus-interesting.md) | Evaluates an idea fairly — positives, negatives, what's interesting | Have a specific idea to evaluate; tempted to accept or reject too quickly |
| [`/s4h-creativity-other-perspectives`](other-perspectives.md) | Thinks genuinely from other people's positions (OPS) | Decision affects others; want to anticipate how they'll respond; in a negotiation |
| [`/s4h-creativity-water-logic`](water-logic.md) | Explores without premature judgment | Early-stage exploration; premature categorization is killing promising directions |
| [`/s4h-creativity-random-entry`](random-entry.md) | Uses an unrelated stimulus to break cognitive fixation | Thoroughly stuck; "I've exhausted my usual thinking"; need a jolt from outside |
| [`/s4h-creativity-provocation`](provocation.md) | Uses an impossible premise as a springboard to new ideas | Need a deliberate jolt into the impossible; "Po:" prefix on a statement |
| [`/s4h-creativity-consider-factors`](consider-factors.md) | Maps every relevant factor before deciding or acting (CAF) | About to act; want a complete pre-flight checklist; don't want to be surprised |

**Not sure which tool?** → Invoke `/s4h-creativity` — it diagnoses your situation and picks the right skill.
