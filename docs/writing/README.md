# ✍️ Writing

> *Every writing problem has a name. Name it, and the fix becomes obvious.*

---

## What this category is

The writing toolkit is a craft toolkit encoding structured diagnosis and production techniques for fiction, non-fiction, and professional writing. Each skill applies a specific methodology to a specific problem type — character design, structural repair, sentence-level editing, rhetorical analysis — rather than offering generic advice. The unifying principle: writing problems are diagnosable. What feels like "the prose isn't working" is almost always one of a small number of identifiable failures, each with a known fix.

---

## When to reach for it

- A character, scene, or plot element is not working and you can't identify why
- A manuscript has continuity errors, POV violations, or logic inconsistencies
- Non-fiction writing buries the point, drifts in tone, or is calibrated for the wrong reader
- A document needs to be restructured, tightened, or elevated
- Professional writing — reports, copy, technical docs, arguments, briefs — needs to be written or improved
- You need to understand what a piece of writing is doing rhetorically

---

## When *not* to reach for it

**Don't use writing skills for communication strategy.** The writing category addresses the craft of the thing itself — how the document is structured, how the sentences are built, how the character is designed. It does not address the broader question of *how to say something given a relationship and a context* — what channel to use, how to frame difficult news, how to manage a tense conversation. That belongs in [Communication](../communication/).

**Don't use writing skills for narrative strategy.** The narrative category addresses how stories frame reality, how to analyse someone else's narrative, and how to construct a frame for a situation. Writing skills address the craft of a specific document or manuscript. If the question is "how should I tell the story of this company," that's a narrative question. If the question is "how do I write the scene where the protagonist confronts her father," that's a writing question.

The nearest neighbor: [Communication](../communication/) — which addresses the strategic and relational dimensions of how you say something, not the craft of the writing itself. Use communication tools when the question is about the relationship and the context; use writing tools when the question is about the document.
